# Neural Extractor V3.0.6

V3.0.6 makes the interface responsive and resizable and promotes Google Chrome to
the primary guided browser for the `YouTube verbinden` workflow, while keeping the
managed-profile isolation and updater guarantees from earlier releases.

- The main window is resizable and adapts its layout to the available space; the
  settings panel is vertically scrollable so every control, including the
  output-folder `Browse` button, stays visible and reachable at smaller window
  sizes.
- The YouTube connection controls adapt to the available width instead of being
  clipped or pushed off-screen.
- Google Chrome is now the primary guided browser. Sign-in and human verification
  run in a dedicated Neural Extractor Chrome profile stored under LocalAppData;
  the user's normal Chrome profile and data are not modified.
- When Chrome App-Bound cookies cannot be read, Neural Extractor falls back to a
  secure managed Firefox profile in the same isolated LocalAppData location.
  Chrome App-Bound v20 cookies remain unsupported by the bundled yt-dlp, so this
  fallback exists specifically for that case.
- Managed-browser sessions recover from a stale profile lock left behind when a
  browser exited uncleanly, without disturbing a browser that is genuinely still
  running.
- Managed-browser process control validates PID plus process-creation identity
  before it acts, so PID reuse or an already-exited process cannot be mistaken for
  a live session.
- The automatic updater path from V3.0.5 remains supported. Manifest generation,
  SHA-256 and file-size verification, backup-before-replacement, transaction-bound
  startup confirmation, and automatic rollback are all preserved.

Real browser login and multi-monitor window behavior still require field testing
on an owner's PC before either is described as proven. The bundled yt-dlp does not
support Chrome App-Bound v20 cookies, and no PO Token provider is bundled, so an
authenticated Chrome session cannot be guaranteed to download every video.

The EXE remains unsigned, so Windows SmartScreen may warn. Download Neural
Extractor only from the official `AegisAI-Dev/NeuralExtractor` GitHub Release and
verify the published checksum when possible.
