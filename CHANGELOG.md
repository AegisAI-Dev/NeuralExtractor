### Added
- Initial release
- YouTube video download functionality
- Thumbnail download support
- Subtitle (SRT) download support
- Batch download support
- Playlist and Mix support
- Cross-platform icon support (Windows .ico, Linux/Mac .png)
- Auto-updater for dependencies
- Modern GUI with navy blue/teal theme

### Features
- Download YouTube videos in various qualities
- Download thumbnails automatically
- Download subtitles in multiple languages
- Batch processing of multiple URLs
- Progress bar and logging
- Cross-platform support (Windows, Linux, macOS)
